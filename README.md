# UNIDAVI - Desenvolvimento Web 2025

## Descrição
Este repositório contém todos os arquivos das atividades realizadas na disciplina **Desenvolvimento Web 1**.  
O objetivo das atividades é fornecer uma base prática e sólida para o desenvolvimento de aplicações web, abordando desde conceitos básicos de HTML até a introdução a JavaScript, PHP e banco de dados.  

O repositório inclui exemplos práticos de:  
- Estruturas HTML e boas práticas de marcação.  
- Formulários e validações de entrada de dados.  
- Manipulação de DOM e eventos com JavaScript.  
- Programação server-side com PHP, sessões e manipulação de dados.  
- Conexão e manipulação de banco de dados (PostgreSQL).  
- Introdução à programação orientada a objetos (POO).  

Todas as aulas foram estruturadas de forma progressiva, permitindo acompanhar a evolução do conhecimento passo a passo.

---

## Índice

- [Aula 1](#aula01) – Introdução ao GitHub e primeiros arquivos  
  Criação do repositório no GitHub e primeiros arquivos de prática. Foco em versionamento e organização de projetos.

- [Aula 2](#aula02) – Estruturas básicas de HTML  
  Desenvolvimento de páginas simples, aprendendo a organizar textos, listas, links e imagens, aplicando boas práticas de semântica.

- [Aula 3](#aula03) – Introdução a formulários HTML  
  Criação de formulários com inputs, selects, radio buttons e checkboxes, aplicando validações básicas e organização visual.

- [Aula 4](#aula04) – Tela de cadastro e introdução ao JavaScript  
  Criação de uma tela de cadastro de usuários com scripts básicos para interatividade, validação de campos e manipulação de eventos.

- [Aula 5](#aula05) – Projetos em JavaScript: calculadoras  
  Desenvolvimento de uma calculadora básica e uma calculadora de média, reforçando lógica de programação, operações matemáticas e manipulação do DOM.

- [Aula 6](#aula06) – Configuração de PHP  
  Preparação do ambiente local PHP, configuração de servidor e integração com o navegador para desenvolvimento server-side.

- [Aula 7](#aula07) – Introdução ao PHP  
  Primeiro contato com PHP, explorando sintaxe, variáveis, operadores e estruturas de controle, integrando PHP em páginas HTML.

- [Aula 8](#aula08) – Prática de PHP com exercícios  
  Consolidação do conhecimento através de exercícios práticos, incluindo loops, funções e manipulação de dados simples.

- [Aula 9](#aula09) – Recursividade e cliente-servidor  
  Introdução à recursividade em PHP e ao conceito de comunicação entre cliente e servidor, explorando funções que chamam a si mesmas.

- [Aula 10](#aula10) – Sessões em PHP  
  Aprendizado sobre criação, armazenamento, acesso e destruição de informações em sessões PHP, entendendo o conceito de estado em aplicações web.

- [Aula 11](#aula11) – Conexão com banco de dados  
  Implementação de conexão com banco PostgreSQL, manipulação de dados e integração com PHP, permitindo consultas, inserções e atualizações.

- [Aula 12](#aula12) – Prova 2  
  Avaliação prática e teórica cobrindo todo o conteúdo aprendido até esta aula, consolidando conceitos de HTML, JavaScript, PHP e banco de dados.

- [Aula 13](#aula13) – Introdução à Programação Orientada a Objetos (POO)  
  Criação e manipulação de objetos em PHP, incluindo definição de classes, propriedades, métodos e encapsulamento, aplicando conceitos fundamentais de POO para organizar e estruturar código de forma modular.
