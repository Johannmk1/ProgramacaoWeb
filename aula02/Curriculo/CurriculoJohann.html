Sobre Mim

Nome: Johann

Profissão: Desenvolvedor de Sistemas

Sou um desenvolvedor especializado em Delphi, MySQL, integração com APIs e otimização de processos corporativos. Trabalho com bancos de dados como SQLite, MySQL e DBISAM, criando soluções robustas e escaláveis para empresas que precisam de performance e confiabilidade.

No meu dia a dia, utilizo FireDAC para acesso a dados, TInject e WppConnect para automação no WhatsApp, além de integrações com APIs REST e manipulação de grandes volumes de dados. Tenho experiência na construção de rotinas de backup, sincronização e migração de dados entre diferentes SGBDs.

  • Integração com WhatsApp (TInject / WppConnect) para automação de atendimento
  • Sincronização de bancos de dados (SQLite, MySQL, DBISAM)

  • Otimização de consultas SQL e refatoração de código legado

Projetos Relevantes
